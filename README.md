# Heart Disease Prediction Using ANN + Interactive Power BI Dashboard  
*Project Title:*  
Heart Disease Prediction Using Artificial Neural Network and Interactive Power BI Dashboard  

*Highlights*  
- Achieved *97.85% training accuracy* & *97% test accuracy* using Deep ANN (TensorFlow/Keras)  
- End-to-end preprocessing pipeline: Pandas, OneHotEncoder, StandardScaler, ColumnTransformer  
- Dataset: health_activity_data.csv (lifestyle + health activity features)  
- Interactive *Power BI Dashboard* (.pbix) with slicers (Gender, Age, Smoker, Diabetic), DAX measures, cards, donut charts & real-time risk visuals  
- Full project report included (Presidency University format)  

*Tech Stack*  
Python • TensorFlow/Keras • Pandas • Scikit-learn • Power BI • Seaborn • Matplotlib  

*Repository Contents*  
- Heart_Disease_ANN.ipynb → Complete model code + evaluation  
- dashboard.pbix → Fully interactive Power BI file  
- Project_Report.pdf → Official university submission  
- screenshots/ → Dashboard & model output images  
- health_activity_data.csv → Dataset  

*Performance Metrics*  
- Accuracy: 97.85% (train), 97% (test)  
- Precision: 92.31% | Recall: 85.71% | F1-Score: 88.89%
